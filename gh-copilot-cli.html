GH Copilot CLI

Use GitHub Copilot from the command line to work inside a trusted folder, ask it to help with coding tasks, and approve any tools it needs to run.

Prerequisite

Install Copilot CLI first, then open a folder containing code you want to work with.

Start a session

Enter copilot, confirm the trusted-directory prompt, and choose whether to remember the folder.

Sign in and prompt

If needed, use /login, then type or speak a prompt for tasks like fixing bugs or building features.

Approve tools

  • Approve commands Copilot wants to run.
  • Reject any action you do not trust.
  • Press Esc to stop a running operation.
